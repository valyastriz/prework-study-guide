HTML

  • HTML is building blocks of the page where we define the content and framework.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element contains metadata, which helps the browser render the page correctly
  • Elements included in the head are not visible to the user, with the exception of the title, which shows up on the browser tab.

CSS

  • CSS defines the style of the page.
  • There are three ways to style a webpage using CSS:
    • Inline CSS
      • This is used when you want to style an element directly in the HTML file
      • Apply to a specific HTML tag, uxing a style attribute with a CSS rule to style a particular page element only.
      • Useful for quick and permanent changes but are less flexible than external and internal style sheets.
    • Internal CSS Style Sheet
      • Embeds an entire stylesheet directly in the HTML file.
      • Contains CSS rules for the webpage in the head section of the HTML file.
      • Rules only apply to THAT page and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code.
      • The CSS rule will apply to all tagged page elements.
    • External CSS Style Sheet
      • Most common way to style a webpage.
      • This is preferred in most cases because it allows the developers to keep all their CSS rules in a separate file, which makes design changes easier.
    • A margin indicates how much space we want around the outside of an element.
    • A padding indicates how much space we want around the content inside an element.

Git

  • Git allows us to works on projects simultaneously with other programmers.
  • Git provides version control, ensuring we don't loose our work and can restore previous versions of code when needed.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • cd foldername: navigates to location
  • cd: goes back one folder

JaveScript

  • We add JavaScript to our HTML structure to make the website interactive.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • A for loop contains three important statements:
    • The first statement determines the starting point for our loop.
    • The second statement is the condition.
    • The final statement is what allows the array to iterate over each time.
  • A function is a set of instructions that tells the program how to perform a certain task.
  • A function MUST BE CALLED in order to run.
