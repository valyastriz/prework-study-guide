Prework Study Guide

HTML

  • HTML is building blocks of the page where we define the content and framework.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element contains metadata, which helps the browser render the page correctly
  • Elements included in the head are not visible to the user, with the exception of the title, which shows up on the browser tab.

CSS

  • CSS defines the style of the page.

Git

  • Git allows us to works on projects simultaneously with other programmers.
  • Git provides version control, ensuring we don't loose our work and can restore previous versions of code when needed.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • cd foldername: navigates to location
  • cd: goes back one folder

JaveScript

  • We add JavaScript to our HTML structure to make the website interactive.
